# apps

Enviroment variables must be set.

Sample **.env** file:
```
POSTGRES_DBNAME=AlbionMC
POSTGRES_USER=admin      
POSTGRES_PASSWORD=Albionmc123?  
POSTGRES_HOST=postgres
POSTGRES_PORT=5432      

GITHUB_CLIENT_ID=xxxxxxxxxxxxxxxxxx
GITHUB_CLIENT_SECRET=xxxxxxxxxxxxxxxxxxxxxxxxx
```

## Run

```sh
docker compose up
```
